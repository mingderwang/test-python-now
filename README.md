# 安裝方式
## 須先安裝 nvm, nodejs, npm
* 參考 https://www.npmjs.com/package/now
```
npm i -g vercel
```
* 須先申請 vercel 帳號 https://vercel.com/login
## 快速部署
```
now
```

```
vercel
```
